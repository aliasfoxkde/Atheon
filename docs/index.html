Atheon

A secret detection GUI that runs anywhere Java runs. No install. No configuration. Just download and go.

Download atheon.jar java -jar atheon.jar  ·  requires Java 17+

What it does

Atheon scans code, config files, and environment variables for leaked API keys, tokens, and credentials. It catches things like AWS access keys, GitHub personal access tokens, Stripe secret keys, OpenAI keys, Slack tokens, and Twilio SIDs before they end up somewhere they shouldn't.

Open the app, type scan, pick a folder. That's the whole workflow.

Mission

Secret detection should be simple enough that anyone can use it — not just developers comfortable with a terminal. Atheon is a self-contained JAR with a clean GUI that runs on Windows, macOS, Linux, and anywhere else a JRE exists. No platform-specific binary. No setup. One file.

It is also built to grow. Adding a new scanner is one Java file. The rest is automatic.
