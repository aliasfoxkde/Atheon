Atheon

A pattern matching engine. Define what you're looking for. Point it at anything. No runtime required.

Download Windows  ·  macOS  ·  Linux  ·  no install

What it is

Atheon is an infinitely extensible platform whose goal is to identify anything you need to find inside text — files, directories, environment variables, or stdin. The engine doesn't care what you're looking for. You define the patterns, it finds the matches.

atheon ./src
atheon --file config.yaml
atheon --env
cat pipeline.log | atheon -

Extend it

The plugin contract is two methods. If you can write a regex or any boolean expression, you can add a pattern. Drop the file in, rebuild, done.

func (p *myPattern) Name() string             { return "my-pattern" }
func (p *myPattern) Matches(line string) bool { return p.re.MatchString(line) }

There is no configuration format, no DSL, no plugin manager. It is just Go code — readable, testable, and version-controlled alongside the engine.
